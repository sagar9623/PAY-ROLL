Instructions
To successfully set up and run the Payroll Management System on your local machine, follow these steps:

Download XAMPP: Begin by downloading and installing XAMPP, a cross-platform web server solution package that includes Apache, MySQL, PHP, and Perl.

Text Editor: Install a text editor such as Notepad++ or Sublime Text 3 for editing code files.

Download Source Code: Download the "Payroll_Management_System" source code.

Extract Files: Extract the downloaded zip file using WinRAR or any other suitable tool. Locate the "Payroll_Management_System" folder.

Copy to Root Directory: Copy the "Payroll_Management_System" folder and paste it into the root directory of your XAMPP installation. This is typically located in the C: drive, D: drive, or E: drive under the htdocs folder.

Database Setup: Open PHPMyAdmin by navigating to http://localhost/phpmyadmin in your web browser. Create a new database named "payroll".

Import SQL File: Locate the "payroll.sql" file inside the "SQL file" folder of the extracted zip package. Import this file into the "payroll" database you created in the previous step.

Run the Script: Open your web browser and navigate to http://localhost/Payroll_Management_System to run the Payroll Management System.
